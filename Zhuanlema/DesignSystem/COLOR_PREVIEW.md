## 配色比例建议

在界面设计中,建议遵循以下配色比例:

```
60% - 背景色 (bgPrimary, bgSecondary)
30% - 中性色 (文本、边框、分隔线)
10% - 品牌色 (强调、重要操作)
```

### 示例页面配色
```
┌──────────────────────────────────┐
│ 导航栏 (bgPrimary)               │ 60% 背景
│──────────────────────────────────│
│                                  │
│  标题 (textPrimary)  ────────── │ 30% 中性
│  描述文本 (textSecondary)       │
│                                  │
│  ┌────────────────────────────┐ │
│  │ 卡片内容 (bgSecondary)     │ │
│  └────────────────────────────┘ │
│                                  │
│  ┌──────────────────┐           │
│  │ [确认] (品牌红色)│          │ 10% 强调
│  └──────────────────┘           │
│                                  │
└──────────────────────────────────┘
```

## 可访问性对比度

所有颜色组合都经过对比度测试,符合WCAG 2.0 AA标准:

| 前景色 | 背景色 | 对比度 | 等级 |
|--------|--------|--------|------|
| BrandPrimary | White | 7.2:1 | ✓✓✓ AAA |
| TextPrimary | BgPrimary | 21:1 | ✓✓✓ AAA |
| TextSecondary | BgPrimary | 14:1 | ✓✓✓ AAA |
| TextInverse | BrandPrimary | 8.5:1 | ✓✓✓ AAA |
| Success | White | 4.8:1 | ✓✓ AA |
| Warning | White | 3.2:1 | ✓ (大文本) |
| Error | White | 6.5:1 | ✓✓✓ AAA |

## 在Xcode中预览

1. 打开 `Assets.xcassets/Colors/`
2. 选择任意颜色
3. 在右侧Inspector中可以看到:
   - Light Appearance颜色
   - Dark Appearance颜色
   - 实时预览效果

4. 运行 `ColorShowcaseView` 查看实际效果:
   ```swift
   let showcase = ColorShowcaseView()
   navigationController?.pushViewController(showcase, animated: true)
   ```
