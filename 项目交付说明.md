### 步骤3: 在新代码中使用

```swift
import UIKit

class MyViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()
        
        // 设置背景色
        view.backgroundColor = ColorPalette.bgPrimary
        
        // 创建按钮
        let button = UIButton()
        button.backgroundColor = SemanticColors.buttonPrimary
        button.setTitleColor(SemanticColors.buttonPrimaryText, for: .normal)
        button.setTitle("立即开始", for: .normal)
        
        // 创建卡片
        let card = UIView()
        card.backgroundColor = SemanticColors.cardBackground
        card.layer.borderColor = SemanticColors.cardBorder.cgColor
        card.layer.borderWidth = 1
        card.layer.cornerRadius = 12
    }
}
```

## 💡 使用建议

### ✅ 推荐做法

1. **统一使用设计系统颜色**
   ```swift
   // ✅ 好
   view.backgroundColor = ColorPalette.bgPrimary
   
   // ❌ 避免
   view.backgroundColor = .white
   view.backgroundColor = UIColor(hex: "#FFFFFF")
   ```

2. **优先使用语义化命名**
   ```swift
   // ✅ 更好 - 语义清晰
   button.backgroundColor = SemanticColors.buttonPrimary
   
   // ✅ 可以 - 但不如语义化
   button.backgroundColor = ColorPalette.brandPrimary
   ```

3. **在Asset Catalog中管理颜色**
   - 颜色修改在一处生效全局
   - 自动支持Dark Mode
   - 设计师可直接在Xcode中调整

4. **遵循交易场景习惯**
   ```swift
   // 中国市场习惯:红涨绿跌
   priceLabel.textColor = isPriceUp ? 
       ColorPalette.tradingUp :    // 红色
       ColorPalette.tradingDown    // 绿色
   ```

### ⚠️ 注意事项

1. **不要硬编码颜色值**
2. **不要手动处理Dark Mode**(颜色会自动适配)
3. **不要仅用颜色区分状态**(考虑色盲用户)
4. **不要过度使用品牌色**(保持视觉平衡)

## 🔧 自定义和维护

### 修改现有颜色

1. 打开 `Assets.xcassets/Colors/`
2. 选择要修改的颜色集
3. 分别设置Light和Dark模式的颜色
4. 保存后自动生效全局

### 添加新颜色

1. 在Asset Catalog中创建新的Color Set
2. 在 `ColorPalette.swift` 中添加定义:
   ```swift
   static let newColor = UIColor(named: "NewColor")!
   ```
3. (可选)在 `SemanticColors.swift` 中添加语义化用途
4. 更新相关文档

### 颜色命名规范

- **基础颜色**: 描述性名称 (`brandPrimary`, `tradingUp`)
- **语义颜色**: 场景+用途 (`buttonPrimary`, `cardBackground`)
- **避免**: 具体颜色名 (`redColor`, `greenButton`)

## 🐛 常见问题

### Q1: 颜色没有显示?
**A**: 确保已在Xcode中Build项目,Asset Catalog需要编译才能生效。

### Q2: Dark Mode颜色不对?
**A**: 检查Asset Catalog中是否为该颜色设置了Dark Appearance的颜色值。

### Q3: 如何预览所有颜色?
**A**: 运行 `ColorShowcaseView`:
```swift
let showcase = ColorShowcaseView()
navigationController?.pushViewController(showcase, animated: true)
```

### Q4: SwiftUI中如何使用?
**A**: 
```swift
// 方式1: 直接使用颜色名称
Color("BrandPrimary")

// 方式2: 桥接到UIColor
Color(UIColor(named: "BrandPrimary")!)

// 方式3: 使用ColorPalette
Color(ColorPalette.brandPrimary)
```

### Q5: 如何检查颜色对比度?
**A**: 使用在线工具:
- [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/)
- [Colorable](https://colorable.jxnblk.com/)
